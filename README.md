### :runner: Dependencies

- python(v3.8)

- node(v14.6)

  

### :office: Build


- front

  ``` bash
  $ cd frontend
  $ npm install // or yarn
  $ npm build // or yarn build
  ```
  

- backend

    ``` bash
    $ cd backend
    $ npm install
    $ npm install nodemon -g
    $ npm start
    ```
